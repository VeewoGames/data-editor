# Data Editor 文档索引

## 概述

这套文档描述 `data-editor` 当前已经实现的产品能力、核心机制和代码结构，目标读者是：

1. 新加入项目的团队成员。
2. 需要快速建立上下文的 AI agent。

文档只描述当前代码已经体现的事实。未来的执行方案、迁移计划和讨论草案统一放到 [`docs/plans/`](./plans/README.md)。
