# 高级筛选组合 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `data-editor` 中实现与 Notion 主语义对齐的高级筛选组合功能，支持顶部普通 chips、唯一高级筛选入口 chip、`AND/OR` 树状组合、同字段多规则，以及 shared view draft/save/reload 一致性。

**Architecture:** 将 `CollectionView.filters` 从旧的单层 `{ op, rules }` 升级为 `{ topLevelRules, advancedRoot }`。顶部普通 chips 与高级筛选树共享同一份 `filters` 状态；执行层改为“顶层普通规则固定 `AND` + 高级树递归求值”；UI 上保留 `ViewFilterBar` 作为 quick filter 入口，新增高级筛选面板负责树编辑。

**Tech Stack:** React, TypeScript, Node test (`node --test`), Playwright, existing shared-view / profile draft infrastructure

- [ ] **Step 3: 最小实现递归执行**

```js
export function applyViewFilters(rows, query, filters, fieldTypes = {}) {
  const normalizedQuery = normalizeText(query).toLowerCase();
  const topLevelRules = Array.isArray(filters?.topLevelRules) ? filters.topLevelRules : [];
  const advancedRoot = filters?.advancedRoot ?? null;
  if (!normalizedQuery && topLevelRules.length === 0 && !advancedRoot) return rows;

  const indexedRows = attachRowIndexes(rows);
  return indexedRows.filter((row) => {
    if (normalizedQuery && !rowMatchesQuery(row, normalizedQuery)) return false;
    if (!topLevelRules.every((rule) => matchRuleNode(row, rule, fieldTypes))) return false;
    return advancedRoot ? matchGroupNode(row, advancedRoot, fieldTypes) : true;
  });
}
```

- [ ] **Step 3: 实现纯逻辑 helper**

```js
export function mergeTopLevelRuleIntoAdvancedRoot(filters, ruleId) {
  const rule = filters.topLevelRules.find((item) => item.id === ruleId);
  if (!rule) return filters;
  const topLevelRules = filters.topLevelRules.filter((item) => item.id !== ruleId);
  const advancedRoot = filters.advancedRoot ?? createAdvancedRoot();
  return {
    topLevelRules,
    advancedRoot: {
      ...advancedRoot,
      children: [...advancedRoot.children, rule],
    },
  };
}
```

```js
export function canCreateChildGroup(filters, groupId) {
  return getGroupDepth(filters, groupId) < 3;
}
```

```js
export function duplicateNodeInAdvancedRoot(filters, nodeId) {
  return updateGroupChildren(filters, nodeId, (children, index, node) => {
    const duplicate = cloneNodeWithFreshIds(node, collectAllFilterNodeIds(filters));
    return [
      ...children.slice(0, index + 1),
      duplicate,
      ...children.slice(index + 1),
    ];
  });
}
```

- [ ] **Step 4: 更新默认 rule 创建 helper，支持新 node 结构**

```js
export function createDefaultFilterRule(field, fieldType, existingRules = []) {
  return {
    kind: "rule",
    id: createFilterId(field, existingRules),
    field,
    operator: fieldType === "Checkbox" ? "is" : "contains",
    value: fieldType === "Checkbox" ? undefined : fieldType === "Multi-select" || fieldType === "Select" || fieldType === "Relation" ? [] : "",
  };
}
```

```js
function createFilterId(field, existingRules = [], usedIds = new Set()) {
  const safeField = field.replace(/\s+/g, "_");
  const baseId = `filter:${safeField}`;
  const occupied = new Set([
    ...existingRules.map((rule) => rule.id),
    ...usedIds,
  ]);
  if (!occupied.has(baseId)) return baseId;
  let index = 2;
  while (occupied.has(`${baseId}:${index}`)) index += 1;
  return `${baseId}:${index}`;
}
```
