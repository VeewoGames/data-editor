# 关联文档字段 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `data-editor` 中实现 `Document` 字段、详情区左侧第三面板、文档索引/读取链路，以及 profile/local 双通路的文档面板状态持久化。

**Architecture:** 以独立字段类型 `Document` 落地文档引用语义，配置层扩展 `documentFiles/documentFields`，读取层补受控 Markdown 索引与正文读取接口，UI 层新增左侧文档面板并复用只读 Markdown 渲染语义。第一阶段只实现只读链路，不引入 Markdown 编辑器和写回协议。

**Tech Stack:** React 18, TypeScript, Node ESM, Vite, Playwright, node:test, `react-markdown`, `remark-gfm`
